# Product Docs

This folder contains a reference pack based on a March 29, 2026 review of Khan Academy Kids and a proposed original product structure for our app.

Recommended reading order:

1. `01-khan-kids-teardown.md`
2. `02-product-blueprint.md`
3. `03-content-system.md`
4. `04-design-system.md`
5. `05-roadmap.md`

Important note:

- These docs are for product learning and planning.
- They should not be used to clone Khan Academy Kids branding, art, characters, copy, or proprietary content arrangements.
- Our goal is to understand why the product works, then build an original app that is calmer, more parent-curated, and better organized for our use case.
